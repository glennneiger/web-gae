USING THE NEW BUZZ & BANTER

To start the New Buzz, simply log onto Minyanville.com
and click the 'Buzz and Banter' link on the left nav.

Once you've arrived at the New Buzz, please
enable Minyanville-only pop-ups in your browser
for the optimal experience.

If you are using Microsoft Internet Explorer,
follow these instructions:

  1. In Internet Explorer, click Tools.
  2. Highlight Pop-up Blocker.
  3. Click on Pop-up Blocker Settings.
  4. In the Address of Web site to allow field type www.minyanville.com.
  5. Click Add, then click close.

If you are using Mozilla Firefox,
follow these instructions:

  1. In Mozilla Firefox, click Tools.
  2. Next, click Options.
  3. Next to the Block Pop-up Windows
    option, click Allowed Sites.
  4. In the Address of Web site field,
    type www.minyanville.com.
  5. Click Allow, then click close.

If you have any questions or problems getting up and running,
please contact MV support at 212.991.9357 or email us at support@minyanville.com.
