xmlns:portals="http://www.minyanville.com/rss/syndication.htm"
Description:

Tickerdata: Tickerdata element is designed to add all the tickers/stocks this content item is discussing into the element.

TickersValue: This element can only be use with tickerdata and caontans single entry of each ticker symbol associated.

Syndications: Syndication element covers all the portals this content is approved for.

Portals: This element can only be use with Syndicatiuons and contains single entry for each portal this content is approved for.
